Числа и вычисления

1. НАТУРАЛЬНЫЕ ЧИСЛА
Натуральные числа — это числа, которые используются для счёта предметов: 1, 2, 3, 4, 5 и так далее. Множество натуральных чисел обозначается символом N.
2. ЦЕЛЫЕ ЧИСЛА
Целые числа — это расширение множества натуральных чисел, включающее все натуральные числа, их отрицательные значения и ноль. Множество целых обозначается символом Z.

Примеры целых чисел: -3, 0, 17, -43
3. ПРИЗНАКИ ДЕЛИМОСТИ ЦЕЛЫХ ЧИСЕЛ
Признаки делимости - это правила, которые позволяют быстро определить, делится ли одно число на другое без остатка.
Рассмотрим основные признаки делимости для целых чисел.

3.2 Признак делимости на 2
Число делится на 2, если его последняя цифра чётная (0, 2, 4, 6, 8).

3.3 Признак делимости на 3.
Число делится на 3, если сумма его цифр делится на 3.

3.3 Признак делимости на 4.
Число делится на 4, если число, образованное его последними двумя цифрами, делится на 4.

3.4 Признак делимости на 5.
Число делится на 5, если его последняя цифра 0 или 5.

3.5. Признак делимости на 6.
Число делится на 6, если оно делится одновременно на 2 и на 3.

3.6. Признак делимости на 9.
Число делится на 9, если сумма его цифр делится на 9.

3.7. Признак делимости на 10.
Число делится на 10, если его последняя цифра 0.

1. РАЦИОНАЛЬНЫЕ ЧИСЛА
Рациональные числа — это числа, которые можно представить в виде дроби a/b, где a - числитель, b - знаменатель, b ≠ 0. Множество рациональных чисел обозначается буквой Q.
2. ДЕСЯТИЧНЫЕ ЧИСЛА
Десятичные числа — это числа, записанные в десятичной системе счисления. Они могут быть:
  • Конечными (например, 0.5, 0.36),
  • Бесконечными (например, 3.1417). Это дроби, у которых после запятой бесконечное множество цифр,
  • Периодическими (например, 1.3333, 0.666). Это дроби, у которых после запятой одна или несколько цифр повторяются. Повторяющаяся часть называется периодом.
3. ОБЫКНОВЕННЫЕ ДРОБИ
Обыкновенные дроби — это запись числа в виде a/b, где a - числитель, b - знаменатель, b ≠ 0. Обыкновенные дроби могут быть:
  • Правильными (числитель больше знаменателя),
  • Неправильными (знаменатель больше числителя).
4. ДЕСЯТИЧНЫЕ ДРОБИ
Десятичная дробь — это дробь, записанная в десятичной системе счисления.
5. ПРОЦЕНТЫ
Процент — это одна сотая часть. Обозначается % (1% = 1/100).
Пример: 50% = 50/100

Чтобы найти процент от числа k, надо разделить процент на 100 и умножить это на k.
Например, надо найти 15 процентов от 90. (15/100) * 90 = 13,5

1. ОПРЕДЕЛЕНИЕ
Логарифм числа b по основанию a — это показатель степени, в которую нужно возвести число a, чтобы получить число b, где a > 0, a ≠ 1, b > 0.
Записывается логарифм следующим образом:
logab = c, где ac = b.
a — основание логарифма,
b — подлогарифмическое выражение,
c — значение логарифма.
2. ДЕСЯТИЧНЫЙ ЛОГАРИФМ
Десятичный логарифм *mdash; это логарифм по основанию 10. Записывается следующим образом: lg(a), где a — какое-то число.
3. НАТУРАЛЬНЫЙ ЛОГАРИФМ
Натуральный логарифм — это логарифм по основанию e (число Эйлера, e ≈ 2.718). Записывается следующим образом: ln(a), где a — какое-то число.

2. СВОЙСТВА АРИФМЕТИЧЕСКИХ ОПЕРАЦИЙ.
  1. Коммутативность сложения и умножения: a + b = b + a; a * b = b * a
  2. Ассоциативность сложения и умножения: (a + b) + c = a + (b + c); (a * b) * c = a * (b * c)
  3. Дистрибутивность умножения относительно сложения: a * (b + c) = a * b + a * c
3. ПРИБЛИЖЁННЫЕ ВЫЧИСЛЕНИЯ.
При работе с действительными числами часто используются приближённые значения, так как точные значения могут быть неизвестны или неудобны для вычислений.

Пример: Вычислите приближённое значение выражения 12.34 + 45.67
12.34 ≈ 12; 45.67 ≈ 46
12 + 46 = 58
Точное значение: 12.34 + 45.67 = 58.01
